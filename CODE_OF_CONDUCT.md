# Code of Conduct

## Our Pledge
We pledge to make participation a harassment-free experience for everyone.

## Our Standards
Be respectful and inclusive.

## Enforcement Responsibilities
Project maintainers are responsible for enforcing standards.

## Scope
This code applies within all project spaces.

## Enforcement
Instances of abuse may be reported.

## Enforcement Guidelines
Warnings, temporary bans, or permanent bans may be applied.

## Attribution
This Code of Conduct is adapted from the Contributor Covenant.
